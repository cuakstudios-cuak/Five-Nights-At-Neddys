# Five-Nights-At-Neddys
¡BIENVENIDOS A ÑEDDYS! (Beta)  Olvida sustos ágiles; aquí el terror pesa. Vigila a robots tan gordos que no corren, ¡ruedan! Si oyes rozar metal y grasa, cierra la puerta o morirás aplastado por un Jumpscare a cámara lenta. PD: Como es una Beta, si el juego va lento es por los polígonos... o los kilos.
